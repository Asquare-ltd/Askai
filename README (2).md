## What is ASK AI?

Most AI resets every conversation. ASK AI doesn't.

ASK AI is built around a **Human Digital Twin** — a living, evolving model of you that captures your emotions, goals, memory, and behavioral patterns. Every session, it knows more about you than the last.

## Architecture

ASK AI is built on a 7-layer Human Digital Twin framework:

1. **Perception Engine** — captures text input and context
2. **Emotion Engine** — detects stressed / motivated / neutral in real time
3. **Human Digital Twin** — maintains your persistent user model
4. **Memory Engine** — stores conversation and mood history
5. **Goal Engine** — tracks multiple goals across life areas
6. **Prediction Engine** — detects burnout risk from mood patterns
7. **Reasoning Engine** — synthesizes all layers into personalized guidance

## Tech Stack

3. Connect your GitHub and select this repo
4. Set main file as `app.py`
5. Click Deploy — live in 2 minutes
